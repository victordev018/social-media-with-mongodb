## Rede Social

Uma API que simula uma rede social, permitindo a criação de novis usuários, criação de posts e até 
mesmo comentar em uma publicação existente.

## ⚙️ Tecnologias envolvidas
- Java
- Spring Boot
- JPA
- MongoDB
